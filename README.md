# crypto-server

## 암호화폐 자동거래 프로그램 서버단

### 설명
업비트 API를 사용한 암호화폐 자동매매를 목적으로 하는 프로그램입니다.
현재는 SpringBatch를 이용한 가격 수집 배치 코드만 작성되어 있습니다.
백엔드와 프론트엔드 프로젝트를 각각 나눠서 개발예정입니다.

### 개발환경
 언어
* Java

 빌드
* Gradle

 프레임워크
* SpringBoot

 DB
* MySQL

 배치
* SpringBatch

 스케쥴링
* Quartz

 메시지
* Slack
